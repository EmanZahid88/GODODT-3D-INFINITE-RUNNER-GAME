# 🎮 3D Endless Runner Game (Godot)

## 📌 Overview
This project is a **3D Endless Runner game** developed using the **Godot Engine**, inspired by popular games like *Subway Surfers*. The objective of the game is to survive as long as possible while avoiding obstacles and collecting points.

The project focuses on implementing core game development concepts, including player controls, obstacle generation, collision detection, and scoring systems.

- GDScript
- Game Design Principles

- 🎨 Simple and optimized 3D environment

## 🎮 Gameplay Description
The player continuously runs forward in an endless environment. The goal is to avoid obstacles by moving left, right, or jumping while collecting points. The game becomes progressively challenging as speed and difficulty increase over time.
